## Getting started

**1. Clone the repository**
```
git clone <your-repo-url>
```

**2. Install dependencies**
```
npm install
```

**3. Run dev server**
```
npm run dev
```

This portfolio was built with [Next.js](https://nextjs.org). It requires Node.js v18.17+.
